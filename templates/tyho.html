Honors & Awards
Best Paper Award, International Conference on Computer Design (ICCD), 2008
Best Project Award, The Art and Science of System Level Design (18766), ECE Department, CMU, Spring 2007
Topic: Communication-Aware Face Detection Using NoC Architecture Implemented an AdaBoost-based face detection algorithm with Network-on-Chip (NoC) architecture. The whole implementation was carried out on a Xilinx FPGA board and verified to be much faster than the software application. Contributed to "Communication-Aware Face Detection Using NoC Architecture," in Proc. of Int'l Conference on Computer Vision Systems (ICVS), pp. 181-189, May 2008.
3rd Place (out of 71 teams), IC/CAD Contest, Ministry of Education, Taiwan, 2003
Topic: Clock Tree Optimization for Useful Skew Developed a tool to reduce the number of paths with setup time violation and the total negative slack by applying clock skew scheduling.
3rd Place (out of 136 teams), Trend Software Programming Contest, Trend Micro Inc., 2002
Topic: Good Neighbors Implemented a network application to divide a file into fragments and distribute them to neighbor computers. The divided file will be salvageable as long as a specified proportion of neighbor computers are online. The contest involved other stages of software development such as design documentation, product-release presentation, and so on.
Honorable Mention (out of 45 teams), IC/CAD Contest, Ministry of Education, Taiwan, 2002
Topic: Identification of the Nearly Un-testable Path Delay Faults Developed a tool to identify as many nearly un-testable path delay faults as possible. A path delay fault is referred to be "nearly un-testable" if there does not exist a robust or non-robust test for this fault.
